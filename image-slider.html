Looping Image Gallery

This is an example of setting up an image gallery to run in a continuous loop using Lucid.

The loop is implemented using a Timer-Block which I modified in the
code to run continuously. This was done in the code by replacing:
"setTimout..." with "myInterval = setInterval..."

The looping is stopped by clicking the "Stop Looping" button
which invokes the code: "clearInterval(myInterval)"
